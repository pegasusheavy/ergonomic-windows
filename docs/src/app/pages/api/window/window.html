Window

Native Windows window creation with trait-based message handling.

use ergonomic_windows::window::*;

Overview

The window module provides a type-safe way to create Windows and handle messages. Implement the WindowHandler trait to receive events like paint, resize, and close. 

use ergonomic_windows::window::*;

struct MyHandler;

impl WindowHandler for MyHandler {
    fn on_paint(&mut self, hwnd: HWND) {
        // Handle paint...
    }

    fn on_close(&mut self, hwnd: HWND) -> bool {
        true // Allow close
    }
}

fn main() -> Result<()> {
    let window = Window::builder()
        .title("My Window")
        .size(800, 600)
        .handler(MyHandler)
        .build()?;

    window.show();
    window.run_message_loop();
    Ok(())
}

WindowHandler Trait

Implement this trait to handle window events. All methods have default implementations that do nothing, so you only need to implement the events you care about. 

pub trait WindowHandler: 'static {
    fn on_create(&mut self, hwnd: HWND) {}
    fn on_destroy(&mut self, hwnd: HWND) {}
    fn on_paint(&mut self, hwnd: HWND) {}
    fn on_close(&mut self, hwnd: HWND) -> bool { true }
    fn on_resize(&mut self, hwnd: HWND, width: u32, height: u32) {}
    fn handle_message(&mut self, msg: u32) -> Option<LRESULT> { None }
}

Events

on_create

Called when the window is created (WM_CREATE).

on_destroy

Called when the window is being destroyed (WM_DESTROY).

on_paint

Called when the window needs to be painted (WM_PAINT).

on_close

Called when the user tries to close the window. Return false to cancel.

on_resize

Called when the window is resized (WM_SIZE).

handle_message

Handle any other window message. Return Some(result) to override default handling.

Window

A created window. Automatically destroyed when dropped.
