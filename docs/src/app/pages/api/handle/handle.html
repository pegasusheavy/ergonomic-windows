Handle Management

RAII wrappers for Windows handles that automatically close when dropped.

use ergonomic_windows::handle::*;

OwnedHandle

A safe wrapper around a Windows HANDLE that automatically closes the handle when dropped. This prevents resource leaks and ensures proper cleanup. 

pub struct OwnedHandle {
    handle: HANDLE,
}

Methods

new(handle: HANDLE) -> Result<Self>

Creates a new OwnedHandle. Returns an error if the handle is null or invalid.

new_allow_null(handle: HANDLE) -> Result<Self>

Creates a new OwnedHandle, allowing null handles (but not INVALID_HANDLE_VALUE).

unsafe new_unchecked(handle: HANDLE) -> Self

Creates an OwnedHandle without validation. Caller must ensure the handle is valid.

as_raw(&self) -> HANDLE

Returns the raw HANDLE for use with Windows APIs.

into_raw(self) -> HANDLE

Consumes the OwnedHandle and returns the raw HANDLE without closing it. Caller is responsible for closing.

try_clone(&self) -> Result<Self>

Duplicates the handle, creating a new independently-owned handle.

Example

use ergonomic_windows::handle::OwnedHandle;

// Wrap a raw handle from a Windows API
let handle = OwnedHandle::new(raw_handle)?;

// Use the handle
some_api(handle.as_raw());

// Handle is automatically closed when `handle` goes out of scope

Note: OwnedHandle implements Drop, so the handle is closed when the struct is dropped. Use into_raw() if you need to transfer ownership without closing.

BorrowedHandle

A borrowed reference to a Windows handle. Does not close the handle when dropped. Useful for passing handles to functions without transferring ownership. 

pub struct BorrowedHandle<'a> {
    handle: HANDLE,
    _marker: PhantomData<&'a ()>,
}

Methods

unsafe new(handle: HANDLE) -> Self

Creates a borrowed handle. Caller must ensure the handle remains valid for lifetime 'a.

from_owned(owned: &'a OwnedHandle) -> Self

Creates a borrowed handle from an owned handle.

as_raw(&self) -> HANDLE

Returns the raw HANDLE.

HandleExt Trait

Extension trait for working with Windows handles. 

pub trait HandleExt {
    fn is_valid(&self) -> bool;
}

Implemented for HANDLE. Returns true if the handle is not null and not INVALID_HANDLE_VALUE.
