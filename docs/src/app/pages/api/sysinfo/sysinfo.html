System Information

Query Windows system information including OS version, processor details, and computer identity.

use ergonomic_windows::sysinfo::*;

OsVersion

Operating system version information.

get() -> Result<OsVersion>

Gets the current OS version.

is_windows_10_or_later(&self) -> bool

Checks if running on Windows 10+.

is_windows_11_or_later(&self) -> bool

Checks if running on Windows 11+ (build 22000+).

display_string(&self) -> String

Returns a human-readable version string.

Fields

major: u32 minor: u32 build: u32 service_pack_major service_pack_minor product_type

ProcessorInfo

System processor information.

processor_info() -> ProcessorInfo

Gets processor information for the system.

Fields

architecture

ProcessorArchitecture enum

processor_count

Number of logical processors

page_size

Memory page size in bytes

allocation_granularity

Virtual memory allocation granularity

ProcessorArchitecture

X86 X64 Arm Arm64 Unknown

Computer Names

hostname() -> Result<String>

Gets the NetBIOS computer name.

dns_hostname() -> Result<String>

Gets the DNS hostname.

dns_domain() -> Result<String>

Gets the DNS domain name.

fqdn() -> Result<String>

Gets the fully qualified domain name.

System Summary

system_summary() -> Result<SystemSummary>

Gets a comprehensive summary of system information.

Example

use ergonomic_windows::sysinfo::system_summary;

let info = system_summary()?;
println!("OS: {}", info.os_version);
println!("Hostname: {}", info.hostname);
println!("CPUs: {}", info.processor.processor_count);
println!("RAM: {} GB", info.memory.total_physical / 1024 / 1024 / 1024);
