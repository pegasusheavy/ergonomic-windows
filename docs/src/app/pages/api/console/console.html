Console I/O

Safe wrappers for Windows console operations including reading, writing, colors, and cursor control.

use ergonomic_windows::console::*;

Console

Represents the Windows console for the current process. Provides methods for reading, writing, and manipulating console properties.

Methods

current() -> Result<Console>

Gets the console for the current process.

alloc() -> Result<Console>

Allocates a new console for the current process.

write(&self, text: &str) -> Result<usize>

Writes a string to the console. Returns bytes written.

write_line(&self, text: &str) -> Result<usize>

Writes a string followed by a newline.

read_line(&self) -> Result<String>

Reads a line from the console (blocking).

clear(&self) -> Result<()>

Clears the console screen.

set_foreground(&self, color: Color)

Sets the text foreground color.

set_background(&self, color: Color)

Sets the text background color.

set_text_attribute(&self, attr: TextAttribute)

Sets both foreground and background at once.

Cursor Control

cursor_position(&self) -> Result<(i16, i16)>

Gets the current cursor position (x, y).

set_cursor_position(&self, x: i16, y: i16)

Sets the cursor position.

set_cursor_visible(&self, visible: bool)

Shows or hides the cursor.

Virtual Terminal (ANSI)

enable_virtual_terminal(&self)

Enables ANSI escape code processing for modern terminal features.

enable_raw_input(&self)

Enables raw input mode (no line buffering or echo).
