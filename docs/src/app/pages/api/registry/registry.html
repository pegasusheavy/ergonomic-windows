Registry

Windows Registry access with typed values and automatic resource management.

use ergonomic_windows::registry::*;

Quick Example

use ergonomic_windows::registry::*;

// Read a string value
let product = get_string(
    RootKey::LOCAL_MACHINE,
    "SOFTWARE\\Microsoft\\Windows NT\\CurrentVersion",
    "ProductName"
)?;

// Open a key for multiple operations
let key = Key::open(
    RootKey::CURRENT_USER,
    "Software\\MyApp",
    Access::READ
)?;

for name in key.values()? {
    println!("Value: {}", name);
}

RootKey

Predefined registry root keys.

RootKey::CLASSES_ROOT

HKEY_CLASSES_ROOT - File associations and COM objects.

RootKey::CURRENT_USER

HKEY_CURRENT_USER - Current user settings.

RootKey::LOCAL_MACHINE

HKEY_LOCAL_MACHINE - System-wide settings.

RootKey::USERS

HKEY_USERS - All user profiles.

RootKey::CURRENT_CONFIG

HKEY_CURRENT_CONFIG - Current hardware profile.

Access

Registry access rights.

Access::READ

Read access.

Access::WRITE

Write access.

Access::ALL

Full access.

Access::WOW64_32 / WOW64_64

Access 32-bit or 64-bit registry view on 64-bit Windows.

Combine access flags with .with(): 

let access = Access::READ.with(Access::WOW64_64);

Value

Registry value types. All common Windows registry types are supported. 

pub enum Value {
    String(String),        // REG_SZ
    ExpandString(String),  // REG_EXPAND_SZ
    MultiString(Vec<String>), // REG_MULTI_SZ
    Dword(u32),            // REG_DWORD
    Qword(u64),            // REG_QWORD
    Binary(Vec<u8>),       // REG_BINARY
}

Methods

as_string(&self) -> Option<&str>

Gets the value as a string (if String or ExpandString).

as_dword(&self) -> Option<u32>

Gets the value as a u32.

as_qword(&self) -> Option<u64>

Gets the value as a u64.

as_binary(&self) -> Option<&[u8]>

Gets the value as binary data.

Key

An opened registry key. Automatically closed when dropped.

Methods

open(root: RootKey, path: &str, access: Access) -> Result<Self>

Opens an existing key.

create(root: RootKey, path: &str, access: Access) -> Result<Self>

Creates or opens a key.

get_value(&self, name: &str) -> Result<Value>

Gets a value from this key.

set_value(&self, name: &str, value: &Value) -> Result<()>

Sets a value in this key.

delete_value(&self, name: &str) -> Result<()>

Deletes a value.

subkeys(&self) -> Result<Vec<String>>

Lists all subkeys.

values(&self) -> Result<Vec<String>>

Lists all value names.

Convenience Functions

get_string(root, path, name) -> Result<String>

Reads a string value.

get_dword(root, path, name) -> Result<u32>

Reads a DWORD value.

set_string(root, path, name, value) -> Result<()>

Sets a string value (creates key if needed).

set_dword(root, path, name, value) -> Result<()>

Sets a DWORD value (creates key if needed).
