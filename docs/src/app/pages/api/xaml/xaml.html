WinRT XAML

Modern Windows UI types and XAML Islands support for declarative, adaptive UI.

Overview

The xaml module provides types and utilities for working with WinRT XAML, the modern UI framework used in UWP and WinUI applications.

Note: Full WinRT XAML support requires a UWP/WinUI app context. For Win32 apps, you can use XAML Islands (Windows 10 1903+) to host XAML controls.

UI Framework Comparison

Layout Types

Thickness

Used for margins, padding, and borders:

use ergonomic_windows::xaml::Thickness;

// Uniform thickness (all sides equal)
let margin = Thickness::uniform(16.0);

// Symmetric (horizontal, vertical)
let padding = Thickness::symmetric(8.0, 4.0);

// All four sides
let border = Thickness::new(1.0, 2.0, 1.0, 2.0); // left, top, right, bottom

// Access individual values
println!("Left: {}, Top: {}", border.left, border.top);

Corner Radius

use ergonomic_windows::xaml::CornerRadius;

// Uniform radius
let rounded = CornerRadius::uniform(8.0);

// Access individual corners
let radius = CornerRadius {
    top_left: 8.0,
    top_right: 8.0,
    bottom_right: 0.0,
    bottom_left: 0.0,
};

Grid Length

use ergonomic_windows::xaml::GridLength;

// Auto-size based on content
let auto = GridLength::Auto;

// Fixed pixel size
let fixed = GridLength::Pixel(200.0);

// Proportional (star) sizing
let star = GridLength::Star(1.0);  // 1*
let double_star = GridLength::Star(2.0);  // 2*

use ergonomic_windows::xaml::{HorizontalAlignment, VerticalAlignment};

// Horizontal alignment options
let h_left = HorizontalAlignment::Left;

// Vertical alignment options
let v_top = VerticalAlignment::Top;
let v_center = VerticalAlignment::Center;

let v_stretch = VerticalAlignment::Stretch; // default

let normal = FontStyle::Normal;   // Upright text

Text Wrapping & Trimming

use ergonomic_windows::xaml::{TextWrapping, TextTrimming};

// Wrapping modes
let no_wrap = TextWrapping::NoWrap;           // Single line
let wrap = TextWrapping::Wrap;                // Wrap at boundaries
let wrap_words = TextWrapping::WrapWholeWords; // Wrap whole words only

// Trimming modes
let none = TextTrimming::None;
let char_ellipsis = TextTrimming::CharacterEllipsis; // "Hello Wor..."
let word_ellipsis = TextTrimming::WordEllipsis;      // "Hello..."
let clip = TextTrimming::Clip;                        // "Hello Wor"

Themes & Visibility

use ergonomic_windows::xaml::{ElementTheme, Visibility, Orientation};

// Theme options
let theme = ElementTheme::Default; // System default
let light = ElementTheme::Light;
let dark = ElementTheme::Dark;

// Visibility
let visible = Visibility::Visible;
let collapsed = Visibility::Collapsed; // Takes no space

// Layout orientation
let horizontal = Orientation::Horizontal;
let vertical = Orientation::Vertical;

ScrollBar Visibility

use ergonomic_windows::xaml::ScrollBarVisibility;

let disabled = ScrollBarVisibility::Disabled; // Never shows
let auto = ScrollBarVisibility::Auto;         // Shows when needed (default)
let visible = ScrollBarVisibility::Visible;   // Always shows
let hidden = ScrollBarVisibility::Hidden;     // Never shows, but scrollable

UI Builder

A fluent API for building UI configurations programmatically: 

use ergonomic_windows::xaml::{UiBuilder, ElementTheme, Thickness};

let ui = UiBuilder::new()
    .theme(ElementTheme::Dark)
    .margin(Thickness::uniform(16.0))
    .padding(Thickness::symmetric(8.0, 4.0));

// Access configured values
println!("Theme: {:?}", ui.theme);
println!("Margin: {:?}", ui.margin);

XAML Islands

XAML Islands allow hosting modern XAML controls in Win32 applications (Windows 10 1903+): 

use ergonomic_windows::xaml::XamlHost;

// Create XAML host for a Win32 window
let host = XamlHost::new(hwnd)?;

// The host can be used to set XAML content
// (Full implementation requires UWP app context)

Requirements: XAML Islands require Windows 10 1903+ and may require application identity (packaged app or identity registration).
