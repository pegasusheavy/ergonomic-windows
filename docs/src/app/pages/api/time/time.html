Time & Timers

High-resolution timers, system time, time zones, and timing utilities.

use ergonomic_windows::time::*;

PerformanceCounter

A high-resolution performance counter using QueryPerformanceCounter. Provides sub-microsecond precision.

new() -> Result<PerformanceCounter>

Creates a new performance counter.

now(&self) -> Result<i64>

Gets the current counter value.

elapsed(&self, start: i64, end: i64) -> Duration

Calculates elapsed time between two counter values.

measure<F, R>(&self, f: F) -> Result<(R, Duration)>

Measures the duration of a closure.

frequency(&self) -> i64

Returns counts per second.

Stopwatch

A simple stopwatch for measuring elapsed time with start/stop/reset functionality.

start_new() -> Result<Stopwatch>

Creates and starts a new stopwatch.

start(&mut self) / stop(&mut self)

Starts or stops the stopwatch.

reset(&mut self) / restart(&mut self)

Resets or restarts the stopwatch.

elapsed(&self) -> Result<Duration>

Gets the elapsed time.

Example

use ergonomic_windows::time::Stopwatch;

let mut sw = Stopwatch::start_new()?;
// ... do work ...
sw.stop()?;
println!("Elapsed: {:?}", sw.elapsed()?);

SystemTime

Windows system time with date and time components.

now_utc() -> SystemTime

Gets the current UTC time.

now_local() -> SystemTime

Gets the current local time.

to_file_time(&self) / from_file_time(u64)

Converts to/from Windows FILETIME format.

day_name(&self) / month_name(&self)

Gets human-readable day/month names.

Fields

year month day day_of_week hour minute second milliseconds

TimeZone

Time zone information including bias, daylight saving status, and names.

current() -> Result<TimeZone>

Gets the current time zone information.

total_bias(&self) -> i32

Gets total UTC offset in minutes.

is_daylight_saving(&self) -> bool

Returns true if currently in DST.

Helper Functions

tick_count() -> u64

Gets milliseconds since system boot (using GetTickCount64).
