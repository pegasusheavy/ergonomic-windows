Installation

Add ergonomic-windows to your Cargo.toml: 

[dependencies]
ergonomic-windows = "0.1"

Requirements

  • Rust: 1.70.0 or later
  • Target: Windows 10/11 (x86_64-pc-windows-msvc)
  • Dependencies: Uses the official windows crate

Basic Usage

Import the prelude to get access to all commonly used types: 

use ergonomic_windows::prelude::*;

fn main() -> Result<()> {
    // Your code here
    Ok(())
}

String Conversions

Windows APIs use UTF-16 strings. ergonomic-windows handles this automatically: 

use ergonomic_windows::string::{to_wide, from_wide, WideString};

// Convert to UTF-16 (null-terminated)
let wide: Vec<u16> = to_wide("Hello, Windows!");

// Convert back to UTF-8
let s = from_wide(&wide)?;

// Use WideString for API calls
let ws = WideString::new("filename.txt");
let ptr = ws.as_pcwstr(); // For Windows API calls

Handle Management

Handles are automatically closed when dropped: 

use ergonomic_windows::handle::OwnedHandle;

{
    let handle = OwnedHandle::new(raw_handle)?;
    // Use the handle...
} // Handle is automatically closed here

Error Handling

All fallible operations return Result<T, ergonomic_windows::Error>. The error type wraps Windows errors and provides helpful context: 

use ergonomic_windows::{Result, Error};

fn do_something() -> Result<()> {
    let key = registry::Key::open(
        RootKey::CURRENT_USER,
        "NonExistent\\Path",
        Access::READ
    )?; // Returns Err if key doesn't exist

    Ok(())
}
