ergonomic-windows

Ergonomic, safe, and efficient wrappers around Windows APIs for Rust. Stop wrestling with raw Win32 calls and start building.

Get Started View on GitHub
Cargo.toml 
[dependencies]
ergonomic-windows = "0.1"

String Conversions

Seamless UTF-8 to UTF-16 conversions with WideString and builder patterns.

RAII Handle Management

Automatic handle cleanup with OwnedHandle. No more leaked resources.

Process Management

Spawn, wait, and manage Windows processes with a familiar builder API.

Registry Access

Read and write registry keys with typed values. Supports all common registry types.

Window Creation

Create native Windows with message handling using a trait-based event system.

File System

Windows-specific file operations including attributes, metadata, and more.

Quick Example

use ergonomic_windows::prelude::*;

fn main() -> Result<()> {
    // String conversions are automatic
    let wide = WideString::new("Hello, Windows! ðŸªŸ");

    // Spawn a process with the builder pattern
    let process = Command::new("notepad.exe")
        .arg("file.txt")
        .spawn()?;

    // Read from the registry
    let version = registry::get_string(
        RootKey::LOCAL_MACHINE,
        "SOFTWARE\\Microsoft\\Windows NT\\CurrentVersion",
        "ProductName"
    )?;

    println!("Running on: {}", version);
    Ok(())
}
